Word Blitz 1.1.0 Update Curtis Thompson Tuesday 30th August 2016 Projects and Programming

Word Blitz has been updated on the Google Play Store today, these are the changes to the game:

  • New achievements and modified two of the achievements to make them easier
  • Nine Letter Practice Mode
  • Minor UI updates

Achievements
Two achievements have been modified. Points required for 'Certified Author' has been brought down from 250,000 to 100,000 while points required for 'Walking Dictionary' has been changed from 500,000 to 250,000. Three more achievements have been added; 'Just For Practice', 'Nine Squared' and 'Practice Makes Perfect'. They require 1, 9 and 50 words in the new Nine Letter Practice mode respectively.

Nine Letter Practice Mode
A new mode to practice finding nine letter words, the most valuable words in the game. A combination of letters is given which is guaranteed to have a nine letter word in, the user must just try to find it. One point is awarded for each nine letter word found. The user has the ability to skip a selection of letters if they find it too hard, as this is just a practice mode.

Minor UI Updates
User is now notified when a new screen is loading after clicking a button on the menu. The 'How To Play' screen has also been updated.
