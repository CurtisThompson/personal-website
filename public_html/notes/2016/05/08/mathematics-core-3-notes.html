Core 3 Notes

Curtis Thompson Sunday 8th May 2016 Academics

Core 3 is the first A2 mathematics module. It expands on a lot of information that should have been learnt in the AS modules, but also introduces three new trigonometric functions and a few rules when differentiating more complex functions. Overall if you are comfortable with the AS modules it should not be too hard to complete this module. You can download my handwritten notes from this page.

Click here to view and download the notes. (2MB)

The topics covered in the notes are:

  • Algebraic Fractions
  • Functions
  • Exponential Functions
  • Numerical Methods
  • Graph Transformations
  • Secant, Cosecant and Cotangent
  • Trigonometric Identities
  • Differentiation Chain Rule
  • Differentiation Product Rule
  • Differentiation Quotient Rule
  • Differentiating Exponential and Logarithmic Functions
  • Differentiating Trigonometric Functions

Remember these are my revision notes. I've sat this exam before and I'm studying all of the further pure modules so these notes may not have enough information for everybody (although I hope they would as you should have learnt everything already and be using these for revision). Practice questions are important too so get started and use these notes when stuck.
